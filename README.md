# Car Rental System (Java)

A console-based Car Rental System developed using Java and Object-Oriented Programming principles.

## Features
- View available cars
- Rent a car
- Return a car
- Prevents double booking
- Case-insensitive car ID selection

## Technologies Used
- Java
- IntelliJ IDEA
- Git & GitHub

├── Main.java

├── CarRentalSystem.java
├── Customer.java
